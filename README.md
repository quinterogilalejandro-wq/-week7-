# Convolución de Matrices Manual
**Semana 7 - Deep Learning**

### Explicación 
* **Convolución:** Es una operación de filtrado que permite extraer características (como bordes de un auto) mediante un kernel.
* **Padding:** Añade un marco de ceros a la imagen para que el resultado mantenga el tamaño original y no se pierda información en los bordes.
* **Stride:** Es el "salto" del filtro. Un stride mayor a 1 reduce el tamaño de la imagen, comprimiendo la información.

### Resultados
El código implementado procesa la imagen de un auto clásico aplicando filtros de detección de bordes y comparando cómo cambian las dimensiones de salida según los parámetros utilizados.
